# JagX AI Platform

## Overview
JagX AI is a comprehensive AI platform that offers chat and coding capabilities, allowing users to enhance their experience by installing custom skills.

## Features
- Authentication
- User Dashboard
- AI Chat and Coding Agent
- API Marketplace
- API Key Management
- Developer Documentation
- Credit System
- Subscription Plans
- Admin Panel
- Mobile Responsive UI
- SEO Optimization
- Vercel Deployment Support
- Supabase Integration

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your environment variables in `.env.local`.
4. Run the development server: `npm run dev`

## Deployment
Deploy your application on Vercel for seamless hosting.
